Before running this project please follow behind steps:
    For cloud connection:
        * change the ip in application.yml
        * change the topic name in application.yml
        * add the ip and hostname to C:\Windows\System32\drivers\etc\hosts

    For email connection:
        Add a file named "passwords.txt" in Data folder. This file should contain the sender's email address in the first line and
        the sender's email password in the second line.
        example:
        xxx@gmail.com
        my_password
