Before running this project please follow behind steps:
    For cloud connection:
        * change the ip in application.yml
        * add the ip and host name to C:\Windows\System32\drivers\etc\hosts

    For email connection:
        Add a file named "password.txt" in Data. This file should contain the sender's email address in the first line and
        the sender's email password in the second line.
        example:
        xxx@gmail.com
        my_password
